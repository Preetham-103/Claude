## 🧩 What's Inside

### 🛠️ Claude Features

#### Skills
Custom skill definitions that extend Claude's behavior for specific tasks. Each skill tells Claude *when* to activate and *how* to respond for a given domain.

| Skill | Purpose |
|---|---|
| `requirement-analyzar` | Breaks down Azure DevOps user stories into actionable requirements |
| `solution-planner` | Converts requirements into implementation-ready technical plans |
| `repo-impact-analyzer` | Maps which modules and services are affected by a change |
| `bug-log-analyzer` | Diagnoses root causes from logs and stack traces |
| `quick-quote-schema-validation` | Validates Quick Quote API payloads |
| `create-quote-schema-validator` | Validates Create Quote API payloads with business logic checks |

#### Artifacts
Standalone interactive tools built inside Claude's artifact system.

| Artifact | Description |
|---|---|
| `Work Tracker` | A persistent task and work-item tracker built as a React artifact |

#### Slash Commands
Custom `/commands` used inside Claude Code to speed up repetitive workflows.

---

### 📚 Projects

#### BookMaven
A fully functional frontend web application built **entirely with Claude Code** as the AI pair programmer. Built with JavaScript, CSS, and HTML — this project was my hands-on exploration of Claude Code's agentic capabilities: how it plans, writes, refactors, and iterates on real code.

**Stack:** JavaScript · CSS · HTML  
**Purpose:** Explore Claude Code end-to-end on a real project

---

### 💡 Prompting Techniques

A growing collection of prompting strategies with before/after examples showing how prompt quality directly impacts Claude's output. Topics covered include:

- Zero-shot vs few-shot prompting
- Chain-of-thought and step-by-step reasoning
- Role prompting and persona setting
- XML structuring for complex outputs
- Prompt chaining for multi-step tasks

## 🚀 Why This Repo Exists

Most people use AI assistants reactively — asking one-off questions and moving on. This repo is about going deeper:

- **Skills** make Claude smarter for my specific work context
- **Artifacts** turn Claude into a tool builder, not just a responder
- **Slash commands** make Claude Code faster and more consistent
- **Projects** prove that Claude Code can ship real software
- **Prompting notes** help me (and others) get better outputs faster
